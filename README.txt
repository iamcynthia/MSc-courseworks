- Computer Programming Coursework 2019/12

- The coursework is to address several type of files such as XML, CSV and JSON files.

- Part1: Pizza Menu 
  --- Change the XML file into illustrative menu

- Part2: Pizza Report 
  --- Write a program to calculate the answers of the questions

- Part3: Pizza Specials
  --- Write a program to change the standard menu into different style

- Part4: Pizza Decider 
  --- Write a program to decide who to get the pizza according to the file pizza_request.json

- Part5: Visualizing Pizza data 
  --- Create a figure containing a subplot, including distplot, boxplot, violin plot and 2D scatter plot created by seaborn and matplotlib
